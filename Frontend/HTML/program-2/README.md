# Resume HTML Page


## Page structure
- Links to external resources (LinkedIn, GitHub, project URLs) are embedded using `[a]` tags.
- The page uses basic HTML tags (`<h1>`, `<h2>`, `<p>`, `<ul>`, `<li>`, `<a>`, `<hr>`, `<pre>`, `<details>`, `<summary>`, `<meter>`, `<font>`) for content structuring and styling.
